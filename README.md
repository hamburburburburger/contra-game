# 魂斗罗 - Contra HTML5

现代重制版 HTML5 魂斗罗游戏，支持双人同屏游玩。

## 功能特性

- 🎮 **双人模式** - 两位玩家同时游玩
- 🔫 **多武器系统** - 普通、散弹、激光、火球四种武器
- 🏃 **横版闯关** - 跑动、跳跃、射击的经典玩法
- 👾 **多种敌人** - 守卫、射手、无人机、Boss
- 🌋 **多样地形** - 平台、桥梁、熔岩坑
- 🎨 **现代画面** - 渐变渲染、粒子特效、视差滚动背景

## 操作说明

### P1 玩家
| 按键 | 动作 |
|------|------|
| A / D | 左右移动 |
| W | 跳跃 |
| S | 趴下 |
| 空格 | 射击 |
| Q | 切换武器 |

### P2 玩家
| 按键 | 动作 |
|------|------|
| ← / → | 左右移动 |
| ↑ | 跳跃 |
| ↓ | 趴下 |
| Enter | 射击 |
| Numpad 0 | 切换武器 |

## 运行方式

直接打开 `index.html` 即可游玩，无需任何服务器。

也可以部署到 GitHub Pages：
1. 将此仓库 fork 到你的 GitHub 账号
2. 在仓库设置中启用 GitHub Pages（选择 main/master 分支）
3. 访问 `https://你的用户名.github.io/contra-game/`

## 技术栈

- HTML5 Canvas
- Vanilla JavaScript
- CSS3
